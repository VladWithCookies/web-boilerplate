# WEB Boilerplate
Boilerplate for modern React apps

## Available Scripts
### Run development server
```bash
npm run dev
```

### Run linters
```bash
npm run lint
```

### Run tests
```bash
npm run test --coverage
```

### Create production build
```bash
npm run build
```

### Serve production build
```bash
npm run start
```
